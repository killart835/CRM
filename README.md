# 🤖 Telegram-бот с Groq AI — Инструкция запуска

## Что умеет бот

- **Авто-режим** — Groq автоматически отвечает клиентам
- **Ручной режим** — сообщения пересылаются менеджеру, он отвечает сам
- **Гибрид** — менеджер может в любой момент нажать «Авто-ответ» под любым сообщением
- Команда `/stats` — статистика диалогов
- Команда `/mode` — переключение режима

---

## Шаг 1 — Создать бота в Telegram

1. Откройте Telegram, найдите **@BotFather**
2. Отправьте `/newbot`
3. Придумайте имя (например: `SmartShop Assistant`)
4. Придумайте username (например: `smartshop_ai_bot`)
5. Скопируйте токен вида `1234567890:ABCdef...` — это `TELEGRAM_TOKEN`

---

## Шаг 2 — Получить ваш Telegram ID

1. Напишите боту **@userinfobot** любое сообщение
2. Он пришлёт ваш `Id` — это `MANAGER_CHAT_ID`

---

## Шаг 3 — Получить ключ Groq

1. Зайдите на [console.groq.com](https://console.groq.com)
2. Зарегистрируйтесь (Google/GitHub, карта не нужна)
3. API Keys → **Create API Key**
4. Скопируйте ключ `gsk_...` — это `GROQ_API_KEY`

---

## Шаг 4 — Деплой на Railway (бесплатно)

1. Зайдите на [railway.app](https://railway.app) и войдите через GitHub
2. Нажмите **New Project → Deploy from GitHub repo**
3. Загрузите файлы бота в GitHub репозиторий (или используйте **Deploy from local** через CLI)
4. В Railway откройте проект → вкладка **Variables** → добавьте:

```
TELEGRAM_TOKEN = ваш_токен
GROQ_API_KEY   = gsk_ваш_ключ
MANAGER_CHAT_ID = ваш_id
GROQ_MODEL     = llama-3.3-70b-versatile
```

5. Railway автоматически запустит бота через `Procfile`

> **Бесплатный план Railway:** 500 часов/месяц — хватит на постоянную работу бота.

---

## Альтернатива: запуск локально (для теста)

```bash
# Установить зависимости
pip install -r requirements.txt

# Создать .env файл
cp .env.example .env
# Отредактируйте .env — вставьте свои ключи

# Запустить
python bot.py
```

---

## Команды бота

| Команда | Кто | Что делает |
|---------|-----|------------|
| `/start` | Клиент | Начинает диалог |
| `/mode` | Менеджер | Переключает авто/ручной режим |
| `/stats` | Менеджер | Показывает статистику |

---

## Как ответить клиенту вручную

1. Бот пересылает вам сообщение клиента с кнопками
2. Нажмите **✍️ Ответить** — бот попросит написать ответ reply'ем
3. Или нажмите **🤖 Авто-ответ** — Groq ответит за вас прямо сейчас
